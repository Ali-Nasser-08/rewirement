# Rewirement

**Rewirement** is a 10-week wellbeing initiative inspired by Yale's "The Science of Well-Being" course. This application helps users track their progress through various challenges designed to increase happiness and build productive routines.

## Features

- **Tactile Retro-Analog Design**: A premium, industrial aesthetic inspired by mid-century design.
- **10-Week Journey**: Comprehensive trackers for all weekly challenges (Signature Strengths, Savoring, Gratitude, Kindness, etc.).
- **Privacy First**: All data is saved locally in your browser. No external servers involved.
- **PWA Support**: Installable as a standalone app on your mobile device or desktop.
- **Multi-Theme**: Choose between Warm Parchment (Light), Dark, and AMOLED Black themes.
- **Progress Reports**: Generate and print a beautiful 3-page A4 PDF report of your journey.
- **PERMA Profiler Integration**: Track your core happiness pillars using the validated PERMA model.

## Installation

This is a static web application. To use it locally, simply open `index.html` in your web browser.

## Deployment

To host this on GitHub Pages:
1. Push the files to a repository.
2. Go to Repository Settings > Pages.
3. Set the source to "Deploy from a branch" and select `main`.
